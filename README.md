# Spor Salonu Yönetim ve Randevu Sistemi

## Proje Hakkında
[cite_start]Bu proje, 2025-2026 Güz Dönemi Web Programlama dersi projesi kapsamında **ASP.NET Core MVC** kullanılarak geliştirilen bir **Spor Salonu (Fitness Center) Yönetim ve Randevu Sistemi**dir[cite: 3, 6].

[cite_start]Sistem; spor salonu hizmetlerini, antrenörlerin uzmanlık alanlarını ve üyelerin randevu süreçlerini yönetmeyi hedefler[cite: 7]. [cite_start]Ayrıca, kullanıcıların fiziksel özelliklerine göre (boy, kilo vb.) **yapay zeka destekli** egzersiz veya diyet önerileri almasını sağlayan bir modül içerir[cite: 7, 31].

## Proje Özellikleri
* [cite_start]**Yetkilendirme:** Admin ve Üye olmak üzere rol bazlı yetkilendirme sistemi (Admin ve Üye girişi)[cite: 49, 51, 52].
* [cite_start]**Randevu Yönetimi:** Üyelerin antrenör müsaitliğine, hizmet türüne ve saate göre randevu alabilmesi ve çakışma kontrolü[cite: 19, 20].
* [cite_start]**Yapay Zeka Entegrasyonu:** OpenAI API veya benzeri bir model ile kişiselleştirilmiş spor/diyet programı önerisi[cite: 30, 31].
* [cite_start]**Yönetici Paneli:** Spor salonu, antrenör ve hizmetlerin tanımlanabildiği, CRUD işlemlerinin yapıldığı yönetim arayüzü[cite: 10, 13, 47].
* [cite_start]**REST API:** Veritabanı ile iletişim sağlayan ve LINQ sorguları ile filtreleme yapan API entegrasyonu[cite: 24, 25].

## Kullanılan Teknolojiler
* [cite_start]**Backend:** ASP.NET Core MVC, C# [cite: 36, 37]
* [cite_start]**Veritabanı:** SQL Server, Entity Framework Core (ORM), LINQ [cite: 38, 39]
* [cite_start]**Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5 [cite: 40, 41]
